为什么写博客

记录技术折腾的过程，方便以后回看，也分享给有需要的人。

技术选型

最终选择了 Astro 作为静态站点生成器，原因：

  • 轻量快速，生成的纯 HTML 加载飞快
  • 原生支持 Markdown 写作
  • Content Collections 让文章管理很方便
  • 部署到 GitHub Pages 零成本

项目结构

gold-bear-blog/
├── src/
│   ├── content/
│   │   ├── config.ts        # 文章 schema 定义
│   │   └── blog/            # Markdown 文章
│   ├── layouts/
│   │   └── BaseLayout.astro  # 全局布局（暗色主题）
│   └── pages/
│       ├── index.astro       # 首页（文章列表）
│       ├── [post].astro      # 文章详情页
│       ├── about.astro       # 关于页
│       └── rss.xml.ts        # RSS 订阅
├── public/
│   └── favicon.svg
├── astro.config.mjs
└── package.json

怎么写文章

在 src/content/blog/ 目录下新建 .md 文件，顶部加上 frontmatter：

---
title: "文章标题"
date: "2026-06-04"
tags: ["标签1", "标签2"]
description: "一句话描述"
---

正文内容...

就是这么简单！
